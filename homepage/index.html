Codeblog API

Codeblog is a free API that allows users to access and manage blogs from anywhere, in any project. With Codeblog, users can easily perform various CRUD (Create, Read, Update, Delete) operations on blogs.

Note : you can fetch blogs of multiples catagories i.e all , latest , coding , webs development , front-end web development and back-end web development

Using Codeblog, you can:

  • Fetch all blogs
  • Fetch a specific blog
  • Create a new blog
  • Update an existing blog
  • Delete a blog
  • Sign up
  • Login

These operations can be performed using simple HTTP requests and responses.

The API is designed to be easy to use, with clear documentation and simple endpoints. With Codeblog, you can save time and effort by integrating blog functionality into your projects quickly and easily.

Get started by checking out the API documentation or by using the buttons below to try out some of the API endpoints:

Get Blogs

Use this endpoint to get a list of blogs by providing catagorie and page number in query

For example : http://localhost:3000/getblogs?catagorie=all&page=1

You can fetch using fetch function :

let res = await fetch( "http://localhost:3000/api/getblogs?catagorie=all&page=1", { method: "GET", headers: { "Content-Type": "application/json", }, } );

let blogs = await res.json();

You can fetch a single blog by providing id of that blog in query of fetch function :

For example : http://localhost:3000/getblogs?id=blogID

You can fetch a single blog by providing id of that blog in query of fetch function :

let res = await fetch( "http://localhost:3000/getblogs?id=blogID", { method: "GET", headers: { "Content-Type": "application/json", }, } );

let blogs = await res.json();

Create Blog

Use this endpoint to create a new blog post.

It will take token for token authentication in headers

It will take title of blog , description of blog and url of image of blog (optional) in body of fetch function

You can fetch a single blog by providing id of that blog in query of fetch function :

For example : http://localhost:3000/createblog

You can fetch a single blog by providing id of that blog in query of fetch function :

let req = await fetch("http://localhost:3000/createblog",{ method : "POST", headers : { "Content-Type" : "application/json", "token" : localStorage.getItem("token") or token }, body : JSON.stringify({ title,description,url }) })

let blogs = await res.json();

Edit Blog

Use this endpoint to edit an existing blog post.

It will take token for token authentication in headers

It will take title of blog , description of blog and url of image of blog (optional) in body of fetch function

You can edit a blog by providing id of that blog in query of fetch function :

For example : http://localhost:3000editblog?id=id

You can fetch a single blog by providing id of that blog in query of fetch function :

let req = await fetch(`http://localhost:3000/editblog?id=id`,{ method : "POST", headers : { "Content-Type" : "application/json", "token" : localStorage.getItem("token") }, body : JSON.stringify({ title,description,url }) })

let blogs = await res.json();

Delete Blog

Use this endpoint to delete an existing blog post.

It will take token for token authentication in headers

You can edit a blog by providing id of that blog in query of fetch function :

For example : http://localhost:3000/delteblog?id=id

You can fetch a single blog by providing id of that blog in query of fetch function :

let req = await fetch(`http://localhost:3000/delteblog?id=id`,{ method : "DELETE", headers : { "token" : localStorage.getItem("token") } })

let blogs = await res.json();

Sign Up

Use this endpoint to Sign up.

You can Sign up in a blog website by providing name , email and password in body of fetch function :

For example : http://localhost:3000/signup

You can fetch a single blog by providing id of that blog in query of fetch function :

let req = await fetch(`http://localhost:3000/signup`,{ method : "POST", headers : { "Content-Type" : "application/json" }, body : JSON.stringify({ name,email,password }) })

let blogs = await res.json();

Login

Use this endpoint to Login.

You can Login in a blog website by providing email and password in body of fetch function :

For example : http://localhost:3000/login

You can fetch a single blog by providing id of that blog in query of fetch function :

let req = await fetch(`http://localhost:3000/login`,{ method : "POST", headers : { "Content-Type" : "application/json" }, body : JSON.stringify({ email,password }) })
